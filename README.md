# mkdesilva.github.io

Personal portfolio site for MK De Silva — full-stack & mobile developer at [Ling](https://ling-app.com), based in Thailand.

Built with React 19, TypeScript, Vite, and Tailwind CSS v4.

## Structure

Multi-entry Vite setup:

- `/` — portfolio (`src/App.tsx`)
- `/nomis/` — Nomis app landing page
- `/nomis/support`, `/nomis/terms`, `/nomis/privacy` — Nomis support pages

## Commands

- `npm run dev` — start dev server
- `npm run build` — typecheck + production build
- `npm run lint` — run ESLint
- `npm run preview` — preview production build
